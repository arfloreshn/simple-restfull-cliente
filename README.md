# Projecto: Simple Cliente CRUD con Jax-RS o RestFull

# Requisitos:

- JDK 1.8 o Java EE8
- Maven
- Primefaces 7.0 Comunitario

# Compatible con los siguiente servidores de aplicaciones
- GlassFish 5.1
- Payara 5.191
- WildFly 17
